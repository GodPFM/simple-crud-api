# Simple CRUD-API

### To the start
    git clone https://github.com/GodPFM/simple-crud-api.git
    git checkout dev
    npm install
### To start app in dev mode
    npm run start:dev
### To start in production mode
    npm run start:prod
### To start in multi mode
    npm run start:multi
### To start tests
    npm run start:dev
    npm run test
### Port
You can change port in .env file

### Endpoint for api -
    /api/users
### Base port - 4000
### Base URL
    http://localhost:4000
